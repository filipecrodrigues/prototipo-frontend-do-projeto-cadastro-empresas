# Protótipo Frontend do Projeto Cadastro de Empresas

Este protótipo simples de frontend desenvolvido para testar o backend do projeto **Cadastro de Empresas Afiliadas** permanecerá neste repositório como um protótipo independente. O objetivo deste protótipo é fornecer uma interface básica para interagir com o backend, que está sendo desenvolvido em Java. Futuramente, o frontend em Angular será criado no repositório do projeto **Cadastro de Empresas Afiliadas**.

## Estrutura do Projeto

A estrutura de diretórios do projeto é a seguinte:

```
index.html
assets/
CSS/
    styles.css
js/
    script.js
```

- **index.html**: Arquivo principal da aplicação.
- **assets/**: Diretório para armazenar recursos adicionais, como imagens.
- **CSS/styles.css**: Arquivo de estilos para a aplicação.
- **js/script.js**: Arquivo JavaScript para a lógica da aplicação.

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

## Objetivo

Este protótipo foi criado para:

1. Testar as funcionalidades do backend desenvolvido em Java.
2. Simular interações básicas com o backend.
3. Servir como base para o desenvolvimento futuro de um frontend em Angular.

## Como Usar

1. Clone este repositório para sua máquina local.
2. Abra o arquivo `index.html` em um navegador para visualizar o protótipo.
3. Certifique-se de que o backend está em execução para testar as funcionalidades de integração.

## Próximos Passos

- Substituir este protótipo por um frontend desenvolvido em Angular.
- Implementar funcionalidades adicionais conforme os requisitos do projeto.
